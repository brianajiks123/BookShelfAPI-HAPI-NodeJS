# Book Shelf APi with HAPI Framework

## Features

- CRUD Books
- Query Books

## Tools

- NodeJS v22.17.1
- HAPI v21.4.0

## Usage

```bash
git clone https://github.com/brianajiks123/BookShelfAPI-HAPI-NodeJS.git
cd BookShelfAPI-HAPI-NodeJS
npm install
npm run start
```

Access with http://localhost:9000/
